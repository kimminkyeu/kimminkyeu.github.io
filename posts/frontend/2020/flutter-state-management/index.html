н”Ңлҹ¬н„° мғҒнғңкҙҖлҰ¬ к°ңл…җ

October 13, 2020

н”Ңлҹ¬н„° мғҒнғңкҙҖлҰ¬ к°ңл…җмқ„ м •лҰ¬н•ҳл Өкі  н•ңлӢӨ.

StatefulWidgetмқҖ мғҒнғңкҙҖлҰ¬лҘј н•ҳкё°мң„н•ң мң„м ҜмқҙлӢӨ.

лӮҳлҠ” riverpodмқ„ нҶөн•ҙ мғҒнғңкҙҖлҰ¬лҘј н•ҳл Өкі  кіөл¶Җ мӨ‘мқҙлӢӨ.

м•„лһҳ мҪ”л“ңмІҳлҹј StatefultWidgetмқ„ мһ‘м„ұн•ңлӢӨ.
мғҒнғңкҙҖлҰ¬ лҗ  ліҖмҲҳл“Өмқ„ м„ м–ён•ҳлҠ” кіімқҖ StateлҘј extendsн•ң classм—җ м ҒлҠ”лӢӨ.


class Test extends StatefulWidget {
  @override
  _TestState createState() => _TestState();
}

class _TestState extends State<Test> {
  _index = 0; // мғҒнғңкҙҖлҰ¬ лҗ  ліҖмҲҳ
  @override
  Widget build(BuildContext context) {
    return Container(
        child: FlatButton(
            child: Text("Hello"),
            onPressed: () {
                setState(() {
                    _index++;
                })
            }
        )
    );
  }
}

мғҒнғң ліҖнҷҳ нҳём¶ң н•ЁмҲҳлҘј нҶөн•ҙ ліҖкІҪлҗң л¶Җл¶„мқ„ мғҲлЎң л°”кҫјлӢӨ.

setState(() {});

мғҒнғңліҖнҷ”лҘј кҙҖлҰ¬н•ҳкі  м•Ңл ӨмЈјлҠ” WidgetмқҙлӮҳ packageлҘј мӮ¬мҡ©н•ҳл©ҙ StatefulWidgetмқҳ мӮ¬мҡ©мқ„ мӨ„мқј мҲҳ мһҲм–ҙліҙмқёлӢӨ.

riverpodм—җлҠ” HookWidgetмқҙ мһҲм–ҙ StatefulWidgetм—ҶмқҙлҸ„ мӮ¬мҡ©н•  мҲҳ мһҲлӮҳліҙлӢӨ.
